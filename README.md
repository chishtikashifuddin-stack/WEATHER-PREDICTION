# Weather Prediction Application

**Admin:** Kashifuddin Chishti

A Python-based application that provides **real-time weather information** using a weather API. This project demonstrates API integration, data handling, and user-friendly output in Python.

## Features

- Fetches current weather data for any city
- Displays temperature, humidity, weather conditions, and more
- Easy-to-use command-line interface
- Handles invalid city names or API errors gracefully
- Can be extended for forecasts or graphical data visualization

## Technologies Used

- Python
- Requests library for API requests
- JSON for data parsing
- Optionally, Tkinter for a GUI interface (if implemented)
